# タイッツー Plus

タイッツー Plus は tampermonkey で動くユーザースクリプトです。
もしかしたら grasemonkey でも動くかもしれないです。

[インストールはこちらから！](https://ikasoba.codesk.dev/taittsuu-plus/index.js)

# タイッツー Plus の機能

- リンク

  URL っぽい文字列をリンクへ変換

- メンションリンク

  `@ikasoba000` などの文字をリンクへ変換

- 画像プレビュー

  `.png`や`.webp`などの拡張子が付いた URL を画像として展開

- リタイーツ
